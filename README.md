# animedb

This is the backend API of "animedb".

A clone of MyAnimeList, Anilist, and Anime DB.

The third entry for Spring Boot projects journey.

Stacks uses:
- Spring Boot (Backend)
 - Java
 - Spring MVC
 - Spring Data MongoDB
 - Spring Security

- React (Frontend)
 - Typescript
 - Material UI

- MongoDB (Database)

- Railway.app (Hosting)
